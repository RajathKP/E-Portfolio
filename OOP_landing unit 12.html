Unit overview

Welcome to Week 12. In this unit, we focus on the practical use of design patterns to structure and organise Python programs in an object-oriented way. Design patterns act as reusable templates that help solve common software design challenges, making code more efficient, modular, and easier to maintain. A key part of this unit involves understanding the factory design pattern and learning how it can be implemented to create objects dynamically.
