Unit overview

In this unit, focus the foundational steps of developing Python programs through object-oriented analysis. This unit explores the understanding how software systems are designed and how user interactions are represented using use case diagrams and state machine diagrams. These tools helped me visualise system behaviour, user roles, and the flow of activities within a program. I also learned about variable management within classes and how data can be retained and shared across instances, which is crucial for building functional and scalable programs. Overall, this unit strengthened my understanding of how to bridge the gap between software design and implementation, setting the stage for effective Python development in later units.

Software Design

This task focus the fundamentals of software design by focusing on understanding user needs and system interactions. The activity involved creating a pen portrait to represent a potential user of an office humanoid robot, helping to visualize how the system would fit into a real-world context. Additionally, I developed a use case model to map out the various ways users might interact with the robot’s software.
