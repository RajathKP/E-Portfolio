Unit 4: Applying a UML Model to a Program Implementation: UML in Practice

.

Unit Overview

In this unit, students explore how to translate UML models—especially class diagrams and activity diagrams—into working Python programs. By doing so, helps to gain practical experience connecting design with implementation, a key skill for professional software development.

Unit Reflection

In this unit focus on applying Unified Modeling Language (UML) principles to software development by transforming design models into executable Python programs.Learn how to interpret and implement UML class and activity diagrams using object-oriented programming (OOP) concepts.

Unit 4 Seminar Title: UML Design

Design an activity diagram which shows the relationships and interactivity between the tasks executed by a humanoid robot performing a task of your choice.  Expand the activity diagram with the development of a class diagram using UML to support a system with basic employee-related functionality which include the retention of employee details and allow an employee to book a day of annual leave.
